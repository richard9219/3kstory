# AI 短剧行业竞品分析

## 国内竞品

### 1. 剪映 / CapCut（字节跳动）
- **定位**：UGC 视频创作工具
- **AI 能力**：AI 配音、AI 字幕、智能剪辑、模板生成
- **优势**：抖音生态加成、海量模板、易上手
- **劣势**：不是专注短剧叙事，更偏快剪工具
- **商业模式**：会员订阅（¥30/月）+ 素材付费

### 2. 腾讯智影
- **定位**：AI 数字人 + 短视频创作平台
- **AI 能力**：文本转视频、数字人播报、智能配音
- **优势**：腾讯云 AI 能力支撑、数字人质量高
- **劣势**：更偏向企业营销视频，短剧叙事能力弱
- **商业模式**：免费试用 + 企业套餐

### 3. 来画 / Laihua
- **定位**：AI 动画短视频平台
- **AI 能力**：手绘风格动画、PPT 转视频、模板库
- **优势**：动画风格独特、适合知识科普类
- **劣势**：真人短剧场景较弱
- **商业模式**：会员制（¥199/年）

## 国外竞品

### 1. Runway (runway.ml)
- **定位**：AI 视频创作专业工具
- **AI 能力**：Gen-2 文本转视频、图像转视频
- **优势**：技术领先、质量高
- **劣势**：价格贵（$12-95/月）
- **商业模式**：订阅制 + 积分制

### 2. Pika (pika.art)
- **定位**：AI 视频生成工具（C 端友好）
- **AI 能力**：文本/图像转 3 秒短视频
- **优势**：界面简洁、速度快
- **劣势**：生成时长短（3-4 秒）
- **商业模式**：免费试用 + 订阅制

## 3kstory 差异化机会

1. **垂直于「短剧」场景**：专注 1-3 分钟竖屏短剧生成
2. **全流程 AI 自动化**：一句话 → 完整短剧
3. **中文短剧生态**：基于 Qwen 的本土化优势
4. **未来 Web3 激励**：创作者积分、NFT、社区分成
